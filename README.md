# Digital Discipliner | Habit Tracker
A lifestyle app to build discipline through daily tracking, visualization, and gamification.


## Overview
==> Helps users track daily habits.
==> Visualises progress with streaks and grids.
==> Gamifies discipline with ranks (Neutral → Legendary).
==> Encourages consistency and mastery.

## Features
- ✅ Daily habit logging
- 📊 Dashboard visualization
- 🔥 Streak tracking
- 🧩 Evolution ranking system
- 📄 Profile management with PDF export

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Visualisation: Grid-based tracker
- Data: Local storage/backend sync
- Export: PDF summaries
